Bem vindo ao projeto

Esta é página do projeto destinada ao público e aos desenvolvedores da página do concurso do IME. Todo o projeto é Software Livre, o que siginifica contribuir sem a esperança de receber algo em troca, mas sim pela satisfação de ajudar a comunidade num todo.

Lembrete !

As aplicações ainda estão em desenvolvimento.

Quais as vantagens de utilizar essas aplicações quando finalizadas?

```O software criado na linguagem Python não esta nos padrões para ser utilizado pelo usuário final devido a uma certa complexidade em sua utilização. Este software será a base para uma interface mais amigável para o usuário.

Já o Script pode ser implementado no código html do próprio IME, mas isso não compete ao usuário, mas sim a Divisão de Telemática do IME caso for conveniente a utilização do script.


A ideia fundamental do projeto é facilitar a obtenção das provas anteriores do concurso do [IME](http://www.ime.eb.br/provas-anteriores-cfg.html). O candidato(a) poderá utilizar essas ferramentas para obeter as provas em formato PDF para a estudar e realizar o concurso. Porém, são muitos arquivos para serem baixados e é nessa hora que as aplicações entram em vigor.




### O que eu preciso para utilizar essas ferramentas?
####Você terá duas formas para usar as ferramentas:

1. Python e suas Bibliotecas
2. Um navegador de internet

#### Navegadores mais conhecidos
Navegador | Link para download

Firefox | www.mozilla.org/firefox-download‎

#### Se você quiser utilizar a versão em [Python](https://www.python.org) segue algumas orientações:
#### Instalar o [Python](https://www.python.org/downloads):


#### Exemplo no Linux Ubuntu

apt-get install python



#### Exemplo no ArchLinux

pacman -S python



#### No Windows
No Windows é necessário fazer download do python pelo site: https://www.python.org/downloads


#### Instalando as dependências
Após ter instalado o [Python](https://www.python.org/downloads) no sistema, é necessário a instalação de uma biblioteca para o funcionamento da aplicação:


#### Instalando no linux

pip3.5 install bs4



#### Executando a aplicação no Linux

$ python download_python_IME.py


#### Sobre a biblioteca [bs4](https://www.crummy.com/software/BeautifulSoup/bs4/doc/)
Um dos motivos para ter utilizado a biblioteca [bs4](https://www.crummy.com/software/BeautifulSoup/bs4/doc/) no software em python é a facilidade de analise de código [html](https://pt.wikipedia.org/wiki/HTML).
Com a biblioteca python [bs4](https://www.crummy.com/software/BeautifulSoup/bs4/doc/), foi possivel realizar uma busca no site das provas anteriores do [IME](http://www.ime.eb.br/provas-anteriores-cfg.html) e baixar os arquivos.

#### Não gostou do [Python](https://www.python.org/downloads)? não tem problema
Ative o [JavaScript](href='https://www.javascript.com/) no seu navegador e rode o [script](https://github.com/MarcosWarlock/IME/blob/master/download_js_IME.js) na página de provas do [IME](http://www.ime.eb.br/provas-anteriores-cfg.html)

#### Código em JavaScript

/* O script deve ser executado na pagina http://www.ime.eb.br/provas-anteriores-cfg.html */
