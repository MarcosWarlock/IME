As aplicações ainda estão em desenvolvimento.

Quais as vantagens de utilizar essas aplicações quando finalizadas?

O software criado na linguagem Python não esta nos padrões para ser utilizado pelo usuário final, devido a uma certa complexidade em sua utilização. Este software será a base para uma interface mais amigável ao usuário.

Já o Script pode ser implementado no código html do próprio IME, mas isso não compete ao usuário, mas sim a Divisão de Telemática do IME caso for conveniente a utilização do script.

Resumindo

A ideia fundamental do projeto é facilitar a obtenção das provas anteriores do concurso do IME. O candidato(a) poderá utilizar essas ferramentas para obter as provas em formato PDF para a estudar e realizar o concurso. Porém, são muitos arquivos para serem baixados, e é nessa hora que as aplicações entram em vigor.

O que é preciso para utilizar essas ferramentas que estão em desenvolvimento?

Há duas formas para usar as ferramentas:

  1. Python e suas Bibliotecas
  2. Um navegador de internet

Para usar a versão em Python segue algumas orientações:

Instalar o Python:

Exemplo no Linux Ubuntu

# apt-get install python

Exemplo no ArchLinux

# pacman -S python

No Windows

No Windows é necessário fazer download do python pelo site: https://www.python.org/downloads

Instalando as dependências

Após ter instalado o Python no sistema, é necessário a instalação de uma biblioteca para o funcionamento da aplicação:

Instalando no linux

# pip3.5 install bs4

Executando a aplicação no Linux

$ python download_python_IME.py

Sobre a biblioteca bs4

Um dos motivos para ter utilizado a biblioteca bs4 no software em python é a facilidade de analise de código html. Com a biblioteca python bs4, foi possivel realizar uma busca no site das provas anteriores do IME e baixar os arquivos.

Não gostou do Python? não tem problema

Ative o JavaScript no seu navegador e rode o script na página de provas do IME

Código em JavaScript

/*
O script deve ser executado na pagina http://www.ime.eb.br/provas-anteriores-cfg.html
*/

var links = document.querySelectorAll('a[href*="pdf"]');
var i;
for (i in links) 
{
    window.open(links[i].href);
}
