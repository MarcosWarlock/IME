Bem vindo ao projeto

Esta é página do projeto destinada ao público e aos desenvolvedores da página do concurso do IME. Todo o projeto é Software Livre, o que siginifica contribuir sem a esperança de receber algo em troca, mas sim pela satisfação de ajudar a comunidade num todo.

Quais as vantagens de utilizar essas aplicações?

A ideia fundamental do projeto é facilitar a obtenção das provas anteriores do concurso do IME. O candidato(a) poderá utilizar essas ferramentas para obeter as provas em formato PDF para a estudar e realizar o concurso. Porém, são muitos arquivos para serem baixados e é nessa hora que as aplicações entram em vigor.

O que eu preciso para utilizar essas ferramentas?

Você terá duas formas para usar as ferramentas:

  1. Python e suas Bibliotecas
  2. Um navegador de internet

Navegadores mais conhecidos

Firefox | www.mozilla.org/firefox-download‎ Google Chrome | https://www.google.com/chrome/browser/desktop/index.html#

Se você quiser utilizar a versão em Python segue algumas orientações:

Instalar o Python:

Exemplo no Linux Ubuntu

# apt-get install python

Exemplo no ArchLinux

# pacman -S python

No Windows

No Windows é necessário fazer download do python pelo site: https://www.python.org/downloads

Instalando as dependências

Após ter instalado o Python no sistema, é necessário a instalação de uma biblioteca para o funcionamento da aplicação:

Instalando no linux

# pip3.5 install bs4

Executando a aplicação no Linux

$ python download_python_IME.py

Não gostou do Python? não tem problema

Ative o JavaScript no seu navegador e rode o script na página de provas do IME

Código em JavaScript

/*
O script deve ser executado na pagina http://www.ime.eb.br/provas-anteriores-cfg.html
*/

var links = document.querySelectorAll('a[href*="pdf"]');
var i;
for (i in links) 
{
    window.open(links[i].href);
}
